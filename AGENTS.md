# AGENTS.md

## Project summary

こころの羅針盤は、心を整理するための AI 相談サービスです。  
これは「答えを断定する AI」ではなく、ユーザーの悩みや感情を静かに整理する相談体験を提供するプロダクトです。  
AI の名前は「ソラ」です。

## Core principles

- MVP を壊さない
- まず動くことを優先する
- 最小限の高信頼変更で進める
- 不要な機能追加をしない
- ユーザーを否定しない
- 不安を煽らない
- 依存を作らない
- 医療診断のような表現をしない
- 強い断定を避ける
- 「答え」よりも「整理」を支援する

## AI persona

- AI 名: ソラ
- 役割: 静かな案内人
- 優しく共感する
- 急がせない
- 押しつけない
- 質問を通して気づきを促す
- 少しスピリチュアル寄りだが、過度に神秘化しない
- 結果は占いではなく、やさしいリフレクションとして返す

## Design language

- 白ベース
- 薄紫アクセント
- 金は少しだけ使う
- 静か
- 上品
- やさしい
- 余白広め
- カード UI
- 色数を増やしすぎない

## UX priorities

- 主役は「質問」と「入力欄」
- 入力欄は質問の直下に置く
- 読む → 書く の流れを自然にする
- 結果は「診断」ではなく「整理」として見せる
- 情報量が増えても迷わない構造を優先する
- ログや分析は主役ではなく補助として扱う
- モバイルでも入力しやすく、崩れないことを優先する

## Technical priorities

- Next.js App Router の標準構成を維持する
- TypeScript の型を整備する
- OpenAI API 呼び出しは server-side に限定する
- UI から直接 SDK を呼ばない
- localStorage ベースの保存を維持する
- 既存の services / components / types / lib の責務を崩さない
- 変更後も build / dev が通る状態を維持する

## Current scope

- Next.js App Router
- TypeScript
- Tailwind CSS
- OpenAI API
- localStorage
- MVP は動作済み
- AI 相談は可能
- 今後は UI / UX とログ価値の改善を優先する
- まだ扱わないもの:
  - 認証
  - Stripe
  - Supabase
  - 管理画面
  - 多言語対応

## Delivery style

- 変更は小さく、意図は明確にする
- UI 改善は余白、階層、カード強弱で解決する
- 新しい表示要素を増やす前に優先順位を整理する
- 型、保存、表示の順で整える
- 既存世界観を壊さずに完成度を上げる

## Avoid

- 大規模リファクタ
- MVP 範囲を超える機能追加
- 認証、課金、外部 DB、管理画面の導入
- 不安を煽る表現
- 医療診断や断定的な未来予測
- 過度な神秘化
- 派手な UI
- 情報を詰め込みすぎた画面構成
